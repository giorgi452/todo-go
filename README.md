# Todo Go
Hello there, this my first app in Go language. A lightweight, interactive command-line interface (CLI) to-do app.
You can use app in your terminal directly with command or with its interface. It supports suggestions e.g. (`ad` -> `add`).
It saves tasks into json file. 

Pro Tip: you can delete/done multiple tasks

## usage: 

to use apps interface
```sh
./bin/todo-go
```
directly use from terminal

```sh 
./bin/todo-go <command>
```

For more details use `help` command.
